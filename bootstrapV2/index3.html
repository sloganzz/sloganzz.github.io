Краткое описание

Модуль KY-008 представляет собой плату простой конструкции с установленным на ней небольшим лазерным диодом в коллиматоре с фиксированным фокусным расстоянием и минимальным набором компонентов для управления током диода. Модуль позволяет реализовывать управление лазером с помощью микроконтроллера, так как токоограничительный резистор на входе модуля не позволит диоду набрать ток, больший, чем может отдать Arduino.

Схема подключения к Arduino

Код поключения к Arduino

int laserPin = 10;
void setup ()
{
pinMode (laserPin, OUTPUT); // define the digital output interface 13 feet
}
void loop () {
laserEmiter(laserPin,15,5);
}
void laserEmiter(int laserPin,int timeToBurn,int timeToRest){
digitalWrite (laserPin, HIGH);
delay (timeToBurn);
digitalWrite (laserPin, LOW);
delay (timeToRest);
}
