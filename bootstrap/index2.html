Датчик вибрации 801S

Краткое описание

становка датчиков вибрации актуальна для проектов в сфере безопасности и автоматизации. Модель 801S отличается высокой чувствительностью и способна реагировать на вибрации с любых направлений в широком диапазоне. Отличительной особенностью сенсора является возможность точной настройки чувствительности сенсора. На выходе датчик измерения вибрации выдает аналоговый и цифровой TTL сигнал, подключается к любому выводу Ардуино. Показания сенсоры актуальны независимо от расположения датчика на рабочей платформе. Датчик вибрации Ардуино предназначен для реализации самостоятельных проектов на базе микросхем этого популярного производителя. Простое подключение для взаимодействия по аналоговому каналу является одним из преимуществ сенсора. В корпусе датчика предусмотрено отверстие для монтажа, через которое можно надежно зафиксировать сенсор на любой платформе. Основные сферы применения датчика – охранные системы, сигнализации, детекторы движения, электронные замки, системы умного дома и домашние сейсмостанции, противоугонные устройства.

Схема подключения к Arduino

Код поключения к Arduino

int vib_pin=7;
int led_pin=13;
void setup() {
pinMode(vib_pin,INPUT);
pinMode(led_pin,OUTPUT);
}

void loop() {
int val;
val=digitalRead(vib_pin);
if(val==1)
{
digitalWrite(led_pin,HIGH);
delay(1000);
digitalWrite(led_pin,LOW);
delay(1000);
}
else
digitalWrite(led_pin,LOW);
}
